# Jarvis
Голосовой ассистент — это Python-приложение, предназначенное для распознавания голосовых команд и выполнения соответствующих действий. Программа работает по принципу "слушай-распознавай-выполняй" и предоставляет пользователю голосовой интерфейс для взаимодействия с компьютером.

Основные цели:

Упрощение повседневных задач через голосовое управление

Автоматизация рутинных операций

Быстрый доступ к информации без использования клавиатуры и мыши

 Основные функции:
 Распознавание голосовых команд на русском языке

 Голосовой ответ (синтез речи)

 Нечеткий поиск команд (распознает команды с ошибками)

 Системные команды
 Получение текущего времени и даты

 Просмотр погоды и новостей

 Поиск в Google, Википедии и Rutube

 Создание скриншотов

 Установка таймера

 Управление громкостью системы

 Выключение компьютера

 Работа с файлами (чтение, создание, открытие)

 Развлекательные функции
 Случайные цитаты ("мудрости")

 Чтение текстовых файлов вслух

 Минимальные требования:
ОС: Windows 10/11 или Linux с поддержкой звука

Python: версия 3.8 или выше

ОЗУ: 2 ГБ

Свободное место: 100 МБ

Микрофон (встроенный или внешний)

Колонки или наушники для голосовых ответов

Принцип работы:
1. Запуск программы
   ↓
2. Инициализация модулей
   ↓
3. Ожидание голосовой команды
   ↓
4. Обработка команды:
   - Очистка от лишних слов
   - Нечеткое сравнение с шаблонами
   - Определение наиболее вероятной команды
   ↓
5. Выполнение действия
   ↓
6. Голосовой ответ
   ↓
7. Возврат к шагу 3

Базовое взаимодействие:
Пользователь: "Джарвис, который час?"
Ассистент: "Сейчас пятнадцать:тридцать"
           (в терминале: полная дата)

Работа с файлами:
Пользователь: "Джарвис, создай файл"
Ассистент: "Введите название файла"
Пользователь: (вводит "заметки")
Ассистент: "Введите содержимое файла"
Пользователь: (вводит текст)
Ассистент: (создает файл "заметки.txt" в Документах)
